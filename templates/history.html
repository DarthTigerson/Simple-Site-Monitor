Search Syntax Examples:

Status Filters
  • status:down - Find logs with down status
  • status:slow - Find logs with slow response
  • status:healthy - Find logs with healthy status
  • status:expiring - Find logs with expiring tokens
Site Filters
  • name:example - Find logs for a specific site name
  • url:example.com - Find logs for specific URLs
  • tag:production - Find logs for sites with a specific tag
Date Filters
  • start_time:2023-05-14 - Logs after this date
  • end_time:2023-05-20 - Logs before this date
Pro Tip: Combine multiple filters like status:down tag:production for more specific results. Use date ranges with start_time:2023-05-14 end_time:2023-05-20.
